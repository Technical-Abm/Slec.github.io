GitHub is a platform that serves as a repository for various projects. On this website, you can discover a wide range of projects. Among them, you will find projects specifically designed for use on Termux or Linux, which are utilized for GitHub scraping, IP information retrieval, and web header extraction.

Projects info

In my GitHub repository, I have three commands that you can run on Termux or Linux. You can utilize the source code from my repository for these three projects. I hope you find these commands beneficial.

Github Scraping

  • pkg update
  • pkg upgrade -y
  • pkg install git -y
  • pkg install python -y
  • git clone https://github.com/Technical-Abm/Github-scraping
  • cd Github-scraping
  • python --version
  • python scraping.py
